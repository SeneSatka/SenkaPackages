# <center> SenkaDiscordColorizer</center>

## Usage

```js
import { colorizer } from "@senka/discord-colorizer";
client.channels.get("<channel_id>").send(
  colorizer.build(`${colorizer.blue("Hello")}
 ${colorizer.red("World")}`)
);
```
